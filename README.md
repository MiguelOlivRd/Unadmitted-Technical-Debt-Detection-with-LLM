# Unadmitted-Technical-Debt-Detection-with-LLM
This repository keeps the code used in my experiment: Trying to use a Large Language Model (Llama 3 8B) to identify Unadmitted Technical Debt using the dataset created in the paper: "Unadmitted Technical Debt: Dataset and Detection Approaches", available at: https://ieeexplore.ieee.org/document/11208161/
